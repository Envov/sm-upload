## how to use?
register an account from https://sm.ms/
```shell
npm i sm-upload -g
sm-upload set-user your-account your-password
sm-upload get-user 
sm-upload ./test.png ./test2.png ./test3.png ...
```
## in typora

```shell
/usr/local/bin/node  /usr/local/bin/sm-upload
```
